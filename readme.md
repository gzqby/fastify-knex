# fastify-knex-ts
Fastify knex connection plugin write by ts.

## Install
```
// this is for Knex of the version is before v1.0
npm i fastify-knex-ts@v1 --save

npm i fastify-knex-ts --save
```
## Usage
```

fastify.register(require('fastify-knex-ts'), [Knex.Config])
```
